#### My Key Projects

*   **[Open Source Contribution/AIClient-2-API](https://github.com/justlovemaki/AIClient-2-API)**:
    AIClient-2-API is a powerful API proxy service. It encapsulates client-only large model capabilities from Gemini CLI, Antigravity, Qwen Code, Kiro, and others into a local OpenAI-compatible interface. Furthermore, it supports account pool management, intelligent polling, automatic failover, and system prompt management.
*   **[Open Source Contribution/OpenClaw-Docker-CN-IM](https://github.com/justlovemaki/OpenClaw-Docker-CN-IM)**:
    OpenClaw-Docker-CN-IM is a Docker image for the integrated Chinese IM plugin version of OpenClaw. It comes pre-installed and configured with plugins for mainstream Chinese IM platforms such as Feishu, DingTalk, QQ Bot, and WeChat Work, enabling rapid deployment of a multi-platform AI bot gateway.
